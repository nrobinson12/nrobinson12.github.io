HBO Enhanced

Web Development - Chrome Extension
Project Overview
What inspired me to develop this lightweight extension was all the extremely prevalent issues that HBO’s streaming service had. I was watching Game of Thrones on HBO Now, and it had many missing features that seemed to be standard on most other streaming sites. As well as some noticeable issues that needed tweaking.
My Contributions
I've added many needed features and fixes, such as changing the grey letterbox to black so that it is less distracting, and adding support for the left and right arrow keys to skip forward and backward. Please check out the Chrome Web Store listing and the Github repository.
Maintenance
Having many people download and try out my extension is a very rewarding feeling. It shows that it was an issue that people found frustrating, and it is super helpful getting feedback on my solution. I plan to keep developing this extension as long as HBO does not fix their site themselves.
